# Phase Evasion

*Phase Evasion* is a fast‑paced, reflex‑driven game built specifically for the *LumenLab*. It combines simple controls with escalating chaos, asking the player to survive as long as possible while navigating an ever‑changing field of hazards and opportunities.

At its core, Phase Evasion is about situational awareness, timing, and restraint. The rules are easy to understand, but surviving the game requires focus and adaptability as the difficulty ramps up.

---

## Summary

You control a color-shifting specter, free to move along the LED strip. Dangerous obstacles (called *flares*) appear, gliding toward you. If you collide with a flare, game over. In addition, bonus gem appear on random spots of the arena, which reward you double points upon capture, assuming you don't collide with any flares.

The longer you survive, the flares become faster and more dense, testing your hand-eye coordination.

#### Primary Objective

Survive for as long as possible earning the highest score you can from flare evasions and gem captures. There is no final level. Phase Evasion ends when you make a mistake. Keyword: *when*.
