# Recall

Recall is a memory and pattern-recognition adaptation of Hasbro’s fifty-year-old classic electronic game *Simon*, built for the *LumenLab*. It challenges the player to observe, remember, and reproduce increasingly complex sequences under pressure.

Unlike reflex-based games, Recall rewards focus and short-term memory. There is no time limit for thinking, but any mistakes are final.

---

## Summary

Recall shows you a randomized sequence of colored lights on the LED strip. After the sequence finishes, it’s your turn to repeat it using the controller. Each successful round adds one more random step to the sequence. The game continues until you mistakenly reply with the incorrect color sequence.

#### Primary Objective

Correctly repeat the LED sequence for as many rounds as possible.

Your score increases with each successfully completed round. There is no maximum level. The sequence grows until your memory gives out.

## How to Play

#### Controls

Rounds swing back-and-forth between the computer and the user, starting with the computer. When the computer displays a multi-colored sequence, respond with the same sequence, pressing the PS3 controller buttons until completion.

* Press △ to input a green color.
* Press ○ to input a red color.
* Press ╳ to input a blue color
* Press ▢ to input a yellow color.

Optionally:
* Press ⇧ (on the D-pad) to speed up the computer sequence replay.
* Press ⇩ (on the D-pad) to slow down the computer sequence replay.

If you guess incorrectly, a red-colored sequence will animate indicating game over. The OLED display will show you your score, and the system high score. Press *Start* to start a new game

## Design Philosophy

Recall is intentionally:

* Calm but tense. No rush, but constant pressure
* Fair. The sequence is always shown clearly
* Mentally demanding. Mistakes come from memory lapses, not surprise

The LED strip becomes a physical memory space, forcing you to visualize patterns rather than rely on muscle memory alone.
